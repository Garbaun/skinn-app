Responsive Height Test

Bu sayfa responsive height testi içindir

Ekran yüksekliğini değiştirerek kutuların nasıl uyum sağladığını gözlemleyin:

  • Geniş Kutu: Yüksekliği 20vh - 50vh arasında değişir
  • Orta Kutu: Yüksekliği 15vh - 30vh arasında değişir
  • Küçük Kutu: Yüksekliği 10vh - 20vh arasında değişir
Büyük Kutu
20vh-40vh
Orta Kutu
15vh-30vh
Küçük Kutu
10vh-20vh

Başarılı Test: Tüm kutular ekran yüksekliğine göre orantılı olarak küçülüp büyüyorsa, responsive height düzgün çalışıyor demektir.
